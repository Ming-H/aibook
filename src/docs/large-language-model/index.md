## 第一章：基础概念

### 1.1 什么是大语言模型
大语言模型是一种基于深度学习的自然语言处理模型，能够理解和生成人类语言。它通过大规模预训练学习语言的统计规律。

### 1.2 发展历程

- T5
- PaLM
