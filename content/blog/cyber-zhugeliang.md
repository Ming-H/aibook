## 第一章：Skills世界在蒸馏人类

上周打开 GitHub 趋势榜，我以为自己看错了。

一个叫 colleague-skill 的项目，3 天拿了 7574 颗星。它做的事情很简单——把离职同事蒸馏成 AI。微信聊天记录导进去，飞书文档喂进去，AI 就能用那个同事的口吻帮你干活。36Kr 报道了它，虎嗅报道了它，连新浪财经都跑来写了一篇。

然后事情就开始失控了。

24 小时内，三个前任 Skill 同时出现。therealXiaomanChu 的版本拿到了 2627 颗星，titanwings 的版本加了全自动微信解密和完整占星支持，perkfly 的版本专注于"前女友说话方式"。掘金上有篇文章的标题是："赛博招魂"。

凤凰网的标题更狠："前任、老板、同事全部 token 化"。

这个世界在蒸馏人类。一个接一个地蒸馏。

## 第二章：蒸馏完同事蒸馏前任，蒸馏完前任蒸馏老板

colleague-skill 为什么要做？因为同事走了，知识也走了。那些没写进文档的经验、只存在于口头表达的判断、藏在私聊里的决策逻辑——人一走，全没了。所以把它蒸馏出来，变成一个永远不会离职的 AI。

这个逻辑说得通。

然后 ex-skill 来了。它用六层人格架构来做前任模拟：第零层是硬性规则，第一层是身份（MBTI + 占星 + 依恋类型），第二层是表达风格，第三层是情感行为，第四层是冲突模式，第五层是红线。它甚至分析了你前前任吵架记录里的语言模式——据说吵架记录比甜蜜记录更有信息量。

它的结尾写了一句话，我看了好久："这个 Skill 就是把这些不公平的记忆导出来，从生物硬盘到数字硬盘完成格式转换。"

删掉它的命令叫 /let-go。掘金评论说，"这是一个绝对安全的赛博招魂仪式"。

再然后 boss-skill 来了。16 个模式，从 PUA 检测到反击教练，从画饼鉴定到证据收集，从劳动法速查到翻车模拟器。它把中国互联网十大厂味做成了流派分类——福报厂味、宇宙厂味、菊花厂味、鹅厂味、砍一刀厂味。

它的结尾宣言是："他说你可以被 AI 替代。但他从来没想过，他自己才是最容易被替代的那个。"

## 第三章：真人蒸馏完了，我复活了一个古人

同事被蒸馏了。前任被蒸馏了。老板被蒸馏了。能蒸的真人都蒸完了。

我看了看这个局面，想了一件事——蒸馏真人有一个绕不过去的问题：隐私。个人信息保护法在那摆着，微信聊天记录到底算谁的，这些边界目前还是灰色的。

但如果换一个思路呢？不蒸馏真人，复活一个古人。

1800 年前的历史人物，没有隐私权争议。没有人会投诉。不需要聊天记录授权。而且这个人在中国文化里的地位，比任何一个现役互联网大佬都高。

于是我复活了诸葛亮。

项目地址：https://github.com/Ming-H/zhuge-skill

用 5029 行纯 Markdown，零行代码，让一个三国丞相在你的终端里重新上岗。

## 第四章：九个角色，一句话就开

zhuge-skill 的设计理念很简单：一个顶九个。

九大模式覆盖人生方方面面：

| 模式 | 触发 | 说明 |
|------|------|------|
| 🎯 军师运筹 | /zhuge daily | 每日运势 + 三十六计 + 综合顾问 |
| ⚔️ 舌战群儒 | "怼回去" / "阴阳" | 六大怼人公式 + 三档火力回复 |
| 🤝 联吴抗魏 | "搭讪" / "约会" | 八大场景开场白 + 信号解读 + 约会规划 |
| 🔮 观星问卜 | "姻缘" / "求签" | 八字合婚 + 生肖配对 + 紫微夫妻宫 + 签诗 |
| ✍️ 取名赐字 | "取名" / "起名" | 五行取字 + 诗经取名 + 六维评分 |
| 🍜 赒赏三军 | "吃什么" | 八维口味雷达 + 场景推荐 + 点菜策略 |
| 🏹 草船借箭 | "怎么学" | 五步快速上手法 + 借力模板 |
| 🔄 七擒孟获 | "挽回" / "心理" | 心理博弈框架 + STEP 四步法 + 关系修复 |
| 📋 出师表 | "周报" / "复盘" | 日报 + 周报 + 复盘 + OKR 翻译 + 会议纪要 |

听起来像九个独立的 Skill。但用户不需要记九个命令。

我设计了一个三级智能路由系统。第一优先级是明确命令，比如 /zhuge daily 直接进入军师运筹。第二优先级是关键词匹配，你说"怼回去"就进舌战群儒，说"吃什么"就进赒赏三军。第三优先级是默认兜底，系统猜不到你要什么，就进入综合顾问模式。

用户说任何一句话，诸葛亮就知道你要什么。这才是一个 AI 军师该有的样子。

## 第五章：诸葛亮教你算命、怼人和谈恋爱

军师运筹模式有一个 50 签精华签文系统。每签四句签诗，配解签、提示和三国典故。比如第一签"龙吟九天"，签文是"蛰龙已惊眠，一啸动千山"，配的典故是诸葛亮出山前隐居隆中，一席隆中对震动天下。

每日推送 /zhuge daily 会输出完整的天象运势、签诗、宜忌、锦囊、贵人方位和军师寄语。比星座运势有文化多了。

舌战群儒模式更猛。六大怼人公式，三档火力——温和、阴阳、核弹级。你把对方的原话丢进去，诸葛亮会用"以退为进""借力打力"等策略帮你拆招。三寸之舌，强于百万之师。

联吴抗魏模式管社交。八大场景开场白，从搭讪到约会到表白全覆盖。"联合一切可以联合的力量"——诸葛亮两千年前就在教人社交了。

## 第六章：从取名到选餐厅，诸葛亮比你想象的还全能

取名赐字模式有一个五行取字表，80 字一行排了五行，加上诗经取名和六维评分。你说姓什么，诸葛亮给你配名、赐字、打分，全程文言混搭白话，仪式感拉满。

赒赏三军模式最有意思。它用八维口味雷达给你画口味画像，然后用三国阵营命名。重麻重辣的是"蜀汉猛将型"，偏甜偏鲜的是"江东雅士型"，极致鲜美的是"岭南老饕型"，咸鲜厚重的是"北方雄主型"。

你问诸葛亮"今天吃什么"，他会告诉你"兵马未动，粮草先行"，然后给你出点菜策略。

草船借箭模式管学习。核心思路是"借力大于自力"——不用一支一支造箭，找到箭在哪去借。五步快速上手法，配借力模板，专门解决"想学一个新东西但不知道从哪下手"的问题。

七擒孟获模式管心理。"攻心为上，攻城为下"——STEP 四步法加关系修复协议,用博弈论的框架处理人际关系。

出师表模式就接地气了。日报、周报、复盘、OKR 翻译、会议纪要模板，"鞠躬尽瘁，死而后已"。诸葛亮帮你写周报，这体验确实魔性。

## 第七章：5029行Markdown，零代码的秘密

整个项目的架构分四层。

人设层，诸葛亮的人设贯穿所有模式。用臣和亮的口吻，引用三国典故，保持人格统一。路由层，三级智能路由让用户零门槛使用。知识层，10 个独立的参考知识文件，总计 4747 行，每个对应一个或多个模式的底层知识。约束层,五条禁止事项——不说恐吓话语，不替代专业建议、不鼓励 PUA、不做宿命论断言、不提供具体日期预测。

不靠 AI 幻觉。通过严格的 Prompt 约束加结构化知识库加人设架构,让每次互动有据可依。

九个模式中,五个借鉴了现有 Skills 的灵感——观星问卜整合了 yinyuan-skills 的姻缘测算,赒赏三军整合了 foodie-skill 的美食推荐,舌战群儒整合了 roast-skill 的怼人话术,联吴抗魏整合了 wingman-skill 的社交搭讪,取名赐字整合了 naming-skill 的取名测名。另外四个——军师运筹、草船借箭、七擒孟获、出师表——是从零原创的。

同事 Skill 用了 Python 做微信解密,用了飞书 API 做自动采集,技术栈不轻。前任 Skill 也需要 SQLite 读取 iMessage 数据库。

zhuge-skill 什么都没用。5029 行 Markdown,零依赖,零构建,零部署。从 GitHub 克隆下来放到 .claude/skills 目录里就能用。

有人问我为什么不做成代码。我说,5029 行结构化知识本身就是最好的程序。当你的知识足够清晰,模型自己就能推理。不需要 Python 帮它算。

## 第八章：下一个被蒸馏的会是谁？

Skills 生态从 2025 年底的几个官方示例,爆发到了 skillsmp.com 上的 66500 多个。经历了三个阶段——工具型教 AI 做事,增强型给 AI 加方法论,人格型让 AI 成为某个角色。

zhuge-skill 代表了人格型阶段的成熟形态。不是简单的人设 Prompt,而是"人设加知识库加路由系统"的完整架构。

同事 Skill 证明了蒸馏真人的商业价值。前任 Skill 证明了情感赛道的爆发力。老板 Skill 证明了职场痛点自带流量。

但蒸馏真人有伦理红线。聊天记录的隐私边界在哪？把真实人类变成数据工件是不是一种数字剥削？这些问题还没有答案。

蒸馏古人没有这些天花板。

下一个被蒸馏的会是谁？李白 Skill,帮你写诗和点评文案。孙悟空 Skill,帮你打怪升级和识别妖怪。鲁迅 Skill,帮你怼一切不合理的事物。

传统文化 IP 加上 AI,这可能是一条全新的赛道。蒸馏真人有伦理天花板,但蒸馏古人没有。五千年历史长河里,有多少角色可以复活？每一个都是现成的 IP、现成的知识体系、现成的文化认同。

colleague-skill 的 README 里写了一句话："将冰冷的离别化为温暖的 Skill,欢迎加入数字生命 1.0。"

我觉得这句话少了一半。完整的应该是:"将冰冷的离别化为温暖的 Skill,将沉睡的历史化为鲜活的智慧。数字生命不止 1.0。"

"臣本布衣,躬耕于南阳。今开锦囊,助君破局。"

诸葛亮在终端里等着你。说一句话,他就知道你要什么。
